Getting Started with SENTRICON

SENTRICON is an industrial safety monitoring platform for cement plants, providing real-time predictive maintenance capabilities.

Key Features:

  • Real-time Monitoring: Stream live sensor data from all critical machinery
  • Alert Management: Intelligent anomaly detection with severity classification
  • Predictive Maintenance: ML-powered health scoring and RUL estimation
  • Role-Based Access: Tailored workflows for operators, engineers, and analysts

Roles & Permissions:

  • Operator: View dashboards, acknowledge alerts, manage basic workflows
  • Safety Engineer: Full monitoring plus safety protocols and reporting
  • Diagnoser: Deep diagnostics, trend analysis, model operations, and feedback analysis
